# Desafio Back End - Java 8

## Requisitos
- Você deve implementar todos o metodos classes em "src\main\java\co\infoprice\challenge\carrinho" conforme a assinatura do método e o javadoc correspondente.
- Não altere os construtores de nenhuma classe, exceto o construtor da classe Carrinho de Compras.
- Não mova as classes de pacote nem como criar novas classes.
- Você pode criar novos métodos privados.

## Dicas
- Utilizamos bastante as features do Java 8 na InfoPrice, principalmente a classe Optional e a lib Stream
- Gostamos de entregas com qualidade e com uma boa cobertura de testes, principalmente unitários.

## Avaliação
- O teste será aprovado se todos métodos funcionarem conforme as regras de negócio especificadas.
- É necessário que o programa esteja fazendo build via maven, e não apenas pela IDE utilizada
